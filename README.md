# lap54: Linux + Apache + PHP 5.4

## Install

```
docker pull clarencep/lap54
```

## Run 

```
docker run -d -v /path/to/your/web/root:/var/www/html -p 8080:80 --name lap54 clarencep/lap54
```

## Exposed Port: 80
